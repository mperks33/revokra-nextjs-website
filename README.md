# Revokra Next.js Website

Production-ready front-end for Revokra using Next.js, TypeScript, Framer Motion, and Lucide icons.

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Deploy to Vercel

1. Create a new GitHub repository.
2. Upload this project's files.
3. Import the repository into Vercel.
4. Vercel will detect Next.js automatically.
5. Click Deploy.

## Included

- Responsive black-and-white SaaS landing page
- Animated dissolving Revokra "R"
- Mobile navigation
- Product dashboard mockup
- Features, use cases, security, and starter pricing sections
- SEO metadata
- Production build scripts
